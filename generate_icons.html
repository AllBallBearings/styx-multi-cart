Styx icons

The extension works without icons (Chrome shows a default puzzle piece). If you'd like a proper toolbar icon, generate the PNGs here, save them into an icons/ folder next to manifest.json, and paste the icons snippet shown below back into manifest.json.

Then add to manifest.json

"action": {
  "default_popup": "popup.html",
  "default_title": "Styx Multi-Cart",
  "default_icon": {
    "16": "icons/icon16.png",
    "32": "icons/icon32.png",
    "48": "icons/icon48.png",
    "128": "icons/icon128.png"
  }
},
"icons": {
  "16": "icons/icon16.png",
  "32": "icons/icon32.png",
  "48": "icons/icon48.png",
  "128": "icons/icon128.png"
}
